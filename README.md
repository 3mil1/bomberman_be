# Bomberman back-end

Application was created for educational purposes

### setPlayer

```json
{
  "method": "setPlayer",
  "args": {
    "name": "playerName"
  }
}
```

### setPlayer to existing server

```json
{
  "method": "setPlayer",
  "args": {
    "name": "player2",
    "roomId": "PUT HERE ROOM ID"
  }
}
```

### startGame

```json
{
  "method": "startGame",
  "args": {
    "roomId": "PUT HERE ROOM ID"
  }
}
```
